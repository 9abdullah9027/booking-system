# Internal Hotel Booking Platform

## Overview
This project is a comprehensive **Internal Booking Platform** designed to streamline hotel operations and management. [cite_start]Originally engineered for **Eleciam LLC**, this solution focuses on reducing booking processing times and providing secure, role-based access for hotel staff and administrators[cite: 25, 26, 29].

[cite_start]The system features a robust **React** frontend integrated with a **PHP/MySQL** backend, facilitating real-time check-in/check-out workflows and efficient guest management[cite: 26, 28].

##  Key Features

###  Security & Access Control
* [cite_start]**Role-Based Access Control (RBAC):** Distinct access levels for Administrators and Staff to ensure data security and operational integrity[cite: 27].
* [cite_start]**Secure Authentication:** implemented secure login and session management protocols for all users[cite: 27].

### Core Operations
* [cite_start]**Check-In/Check-Out Workflows:** Dedicated REST API endpoints to handle guest arrival and departure logic seamlessly[cite: 28].
* [cite_start]**Admin Dashboard:** A centralized hub for viewing occupancy, managing bookings, and overseeing hotel performance[cite: 27].
* [cite_start]**CRUD Operations:** Full Create, Read, Update, and Delete capabilities for bookings, guest profiles, and room inventory[cite: 28].

###  Performance
* [cite_start]**Optimized API Architecture:** Custom-built REST API endpoints that reduced booking processing time by **40%**[cite: 29].

##  Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/9abdullah9027/your-repo-name.git](https://github.com/9abdullah9027/your-repo-name.git)
    ```
2.  **Frontend Setup**
    ```bash
    cd client
    npm install
    npm start
    ```
3.  **Backend Setup**
    * Import the provided `database.sql` file into your MySQL server.
    * Configure your `db_connection.php` file with your local credentials.
    * Serve the PHP folder using XAMPP/WAMP or a built-in PHP server.
